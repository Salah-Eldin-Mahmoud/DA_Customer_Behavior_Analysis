Data Analytics Project
Overview

This project demonstrates a complete end-to-end data analytics workflow using Python, SQL, and Power BI. The goal of the project is to collect insights from raw data through data cleaning, exploratory data analysis (EDA), SQL querying, visualization, and reporting.

The project covers the full analytics process starting from importing the dataset to building an interactive dashboard and presenting business insights.

Dataset

The dataset used in this project contains structured data related to business operations, sales, customers, products, or any relevant domain.

Main dataset tasks:

Loading and exploring the dataset
Handling missing values and duplicates
Data transformation and preprocessing
Feature understanding and analysis
